## ArduFMPLL ##

Arduino FM PLL TSA5511 / SAA1057

### TSA5511 ###

This chip is communcating true i2c.
I got it working with the DRFS 6 v2.0 3Meter FM Transmitter 6Watt.<br>
This pcb you can buy it from https://dutchrfshop.nl/nl/.
